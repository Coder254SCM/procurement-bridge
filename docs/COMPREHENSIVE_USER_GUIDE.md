## üöÄ Quick Start for Beginners {#quick-start}

### What is the Kenya e-GP Platform?

The Kenya e-GP Platform is a **complete digital transformation solution** for government and private procurement. Think of it as:

- **WhatsApp for Procurement**: Easy to use, secure messaging
- **Banking App Security**: Multi-layered protection for your data
- **Amazon Shopping Experience**: Simple, fast, and reliable
- **Blockchain Verification**: Like digital signatures that can't be forged

### Who Can Use This Platform?

1. **Government Agencies** (Ministries, Counties, Parastatals)
2. **Private Companies** (Any size, any industry)
3. **Individual Suppliers** (Small businesses, consultants)
4. **Evaluators & Auditors** (Independent professionals)

### First-Time User Journey (15 Minutes)

#### Step 1: Access the Platform (2 minutes)
```
1. Open your web browser
2. Go to: https://egp.co.ke
3. You'll see a blue "Get Started" button
4. Click it to begin registration
```

#### Step 2: Choose Your Role (1 minute)
You'll see these options:
- **üèõÔ∏è Government Entity**: For ministries, counties, agencies
- **üè¢ Private Organization**: For companies, NGOs
- **üë§ Individual Supplier**: For consultants, small businesses
- **‚öñÔ∏è Evaluator**: For independent evaluation professionals

**üí° Tip**: Choose the option that best describes your organization

#### Step 3: Basic Registration (5 minutes)
```
For Government Entities:
1. Enter ministry/agency name
2. Provide official email (@government.go.ke preferred)
3. Upload authorization letter (PDF, max 5MB)
4. Provide contact person details
5. Create strong password (8+ characters)

For Private Organizations:
1. Company/organization name
2. Business registration number
3. Official company email
4. Upload business certificate
5. Contact person details
6. Create password

For Individual Suppliers:
1. Full name (as on ID)
2. ID/Passport number
3. Mobile number
4. Email address
5. Professional category (construction, IT, consulting, etc.)
6. Create password
```

#### Step 4: Email Verification (2 minutes)
```
1. Check your email inbox
2. Find email from "Kenya e-GP Platform"
3. Click "Verify Email" button
4. You'll be redirected to login page
5. Enter your credentials to access dashboard
```

#### Step 5: Complete Profile Setup (5 minutes)
```
1. Upload profile photo (optional but recommended)
2. Fill in additional business details
3. Upload required documents:
   - Certificate of incorporation
   - Tax compliance certificate
   - Bank details form
4. Set notification preferences
5. Complete verification wizard
```

## üë• User Role Detailed Guides {#user-roles}

### üèõÔ∏è Government Buyer Complete Guide

#### Daily Workflow Overview
As a government buyer, your typical day involves:
1. **Morning**: Review pending tenders and approvals
2. **Mid-morning**: Create new procurement requests
3. **Afternoon**: Evaluate submitted bids
4. **End of day**: Generate reports and communicate with suppliers

#### Creating Your First Tender (30 minutes)

**Step 1: Access Tender Creation**
```
1. Login to your dashboard
2. Click "Create New Tender" (green button, top right)
3. Choose tender type:
   - Goods procurement
   - Services procurement
   - Works/construction
   - Consultancy services
```

**Step 2: Basic Information**
```
Title: Be specific and clear
Example: "Supply of Office Furniture for Ministry of Education Headquarters"

Description: Include:
- What you're procuring
- Quantity needed
- Quality specifications
- Delivery timeline
- Special requirements

Budget Range:
- Enter estimated value
- Choose currency (KES/USD/EUR)
- Set budget visibility (public/private)
```

**Step 3: Detailed Requirements**
```
Technical Specifications:
1. Click "Add Specification"
2. Enter requirement (e.g., "Office chairs must be ergonomic")
3. Set whether it's mandatory or preferred
4. Add evaluation weight (how important it is)
5. Repeat for all requirements

Documents Required from Suppliers:
‚òëÔ∏è Certificate of incorporation
‚òëÔ∏è Tax compliance certificate
‚òëÔ∏è Professional insurance
‚òëÔ∏è Similar work experience
‚òëÔ∏è Financial statements (last 2 years)
‚òëÔ∏è Bank guarantee/performance bond

Evaluation Criteria:
- Technical capability (40%)
- Financial capacity (30%)
- Past experience (20%)
- Price competitiveness (10%)
```

**Step 4: Timeline Setup**
```
Tender Publication Date: [Auto-filled with today]
Clarification Deadline: [7 days from publication]
Submission Deadline: [21 days from publication]
Evaluation Period: [14 days after submission]
Award Date: [7 days after evaluation]

üí° System automatically calculates minimum timelines per PPRA regulations
```

**Step 5: Review and Publish**
```
1. Preview tender document (PDF will be generated)
2. Check all information for accuracy
3. Ensure compliance indicators are green
4. Click "Publish Tender"
5. System will automatically:
   - Post to PPIP portal
   - Send to registered suppliers
   - Create evaluation committee
   - Set up automatic notifications
```

#### Managing Supplier Inquiries

**Clarification Process:**
```
1. Suppliers submit questions through platform
2. You receive email notification
3. Access "Tender Management" ‚Üí "Clarifications"
4. Review question and prepare response
5. Response automatically shared with ALL interested suppliers
6. Amendment notices generated if changes needed
```

**Best Practices:**
- Respond within 3 business days
- Be clear and specific
- If major changes needed, consider extending deadline
- All clarifications are public for transparency

#### Bid Evaluation Process

**Phase 1: Administrative Compliance (Day 1-2)**
```
1. Platform automatically checks:
   - All required documents submitted
   - Submission before deadline
   - Digital signatures valid
   - File formats correct

2. Manual verification needed:
   - Document authenticity
   - Registration validity
   - Tax compliance status

3. System generates compliance report
4. Non-compliant bids automatically filtered out
```

**Phase 2: Technical Evaluation (Day 3-8)**
```
1. Assign evaluators (if not done automatically)
2. Each evaluator receives secure evaluation portal
3. Blind evaluation (no pricing information visible)
4. Score each criterion 1-10
5. System calculates weighted scores
6. Consensus meeting for major discrepancies
7. Generate technical evaluation report
```

**Phase 3: Financial Evaluation (Day 9-10)**
```
1. Only technically compliant bids proceed
2. Financial analysis includes:
   - Price competitiveness
   - Cost breakdown reasonableness
   - Financial capacity assessment
   - Value for money calculation

3. Combined scoring:
   Technical Score √ó Technical Weight + Financial Score √ó Financial Weight
```

**Phase 4: Final Award Decision (Day 11-14)**
```
1. Review top 3 recommended suppliers
2. Conduct due diligence (automated)
3. Check blacklist status
4. Verify references
5. Final approval workflow
6. Award letter generation
7. Unsuccessful bidder notifications
```

## üÜò Troubleshooting & Support {#support}

### Common Issues and Solutions

#### Login and Access Issues

**Cannot Login - Password Issues:**
```
Symptoms:
- "Invalid password" error
- Account locked message
- Password reset not working

Solutions:
1. Check Caps Lock status
2. Clear browser cache and cookies
3. Use password reset option:
   - Click "Forgot Password"
   - Enter registered email
   - Check spam/junk folder
   - Use reset link within 15 minutes
   - Create new strong password

4. If still issues:
   - Contact administrator
   - Verify account status
   - Check email address spelling
   - Request manual password reset
```

**Account Access Problems:**
```
Account Locked:
- Wait 30 minutes for automatic unlock
- Contact administrator for immediate unlock
- Review recent login attempts
- Change password after unlock

Account Suspended:
- Contact system administrator
- Review suspension reason
- Provide required documentation
- Wait for reactivation confirmation

Two-Factor Authentication Issues:
- Use backup codes provided during setup
- Resync authenticator app time
- Contact support for code reset
- Temporarily disable and re-enable 2FA
```

#### Document Upload Issues

**File Upload Failures:**
```
Common Causes and Solutions:

File Size Too Large:
- Maximum size: 10MB per file
- Compress PDF files using online tools
- Split large documents into sections
- Use image compression for photos

Unsupported File Format:
- Supported formats: PDF, DOC, DOCX, JPG, PNG
- Convert files to supported formats
- Avoid ZIP or RAR files
- Use PDF for multi-page documents

Internet Connection Issues:
- Check internet stability
- Try uploading during off-peak hours
- Use wired connection if possible
- Clear browser cache before upload

Browser Compatibility:
- Use latest Chrome, Firefox, or Safari
- Enable JavaScript and cookies
- Disable popup blockers temporarily
- Try incognito/private browsing mode
```

**Document Processing Delays:**
```
Virus Scanning:
- System scans all uploads for viruses
- Large files take longer to process
- Wait for "Upload Complete" confirmation
- Refresh page if stuck for >5 minutes

OCR Processing:
- Text extraction from images takes time
- PDF files process faster than images
- Complex documents need more time
- Check processing status in dashboard

Blockchain Verification:
- Document hashing for integrity
- Network congestion may cause delays
- Verification usually completes in 2-5 minutes
- Contact support if >30 minutes
```

#### Performance and Speed Issues

**Slow Loading Times:**
```
Browser Optimization:
1. Clear browser cache and cookies
2. Close unnecessary browser tabs
3. Disable browser extensions temporarily
4. Update browser to latest version
5. Restart browser completely

Internet Connection:
1. Test internet speed (minimum 5 Mbps)
2. Switch to wired connection if using WiFi
3. Close other internet-heavy applications
4. Contact ISP if persistent issues

System Optimization:
1. Restart computer/device
2. Close unnecessary applications
3. Check available disk space (minimum 1GB)
4. Run system updates
5. Scan for malware/viruses
```

**Page Loading Errors:**
```
Error 500 (Server Error):
- Temporary server issue
- Refresh page after 2-3 minutes
- Clear browser cache
- Try different browser
- Contact support if persistent

Error 404 (Page Not Found):
- Check URL spelling
- Use navigation menu instead of direct links
- Clear browser cache
- Report broken links to support

Timeout Errors:
- Slow internet connection
- Server overload during peak hours
- Large file processing delays
- Try again during off-peak hours
```
